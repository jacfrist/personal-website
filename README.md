# Portfolio Website  

This is my personal portfolio website, built with the help of **Lovable AI** and refined with my own coding skills. It highlights my projects, background, and interests.  

## Features  
- Clean, responsive design  
- Sections for projects, CV, extracurriculars, and contact  
- Built for easy updates and expansion  

## Tech Stack  
- **React / Vite** for frontend  
- **Tailwind CSS** for styling 

## Purpose  
This site serves as a central place to showcase my work and connect with potential employers or collaborators.
